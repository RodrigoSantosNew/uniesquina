# 🎓 UniEsquina

Projeto acadêmico desenvolvido para a disciplina de Scrum e LGPD.

A aplicação simula uma plataforma de cursos online, permitindo cadastro de usuários, autenticação e gerenciamento básico de cursos, aplicando conceitos de desenvolvimento web, versionamento e metodologias ágeis.

---

# 🚀 Tecnologias Utilizadas

- Java
- Jakarta Servlet
- HTML5
- CSS3
- MySQL
- JDBC
- Apache Tomcat 10
- Maven
- Git e GitHub
- Jira

---

# 📋 Metodologia

O projeto está sendo desenvolvido utilizando:

- Scrum

- Backlog de Histórias de Usuário
- Jira para gerenciamento das tarefas
- GitHub para controle de versão

# 👥 Histórias de Usuário Implementadas

## HU01 – Cadastro de usuário

Como visitante, quero me cadastrar na plataforma, para acessar os cursos.

Dados:
- nome
- e-mail
- senha

Base legal:
- execução de contrato

Regras:
- não coletar CPF ou dados desnecessários

---

## HU02 – Login

Como usuário, quero fazer login, para acessar minha conta com segurança.

Dados:
- e-mail
- senha

Base legal:
- execução de contrato

Regras:
- autenticação obrigatória

# 🔐 LGPD

O projeto segue princípios básicos da LGPD:

- Coleta mínima de dados
- Uso apenas de informações necessárias
- Controle de autenticação
- Planejamento de armazenamento seguro de senha
- Consentimento para marketing será implementado nas próximas Sprints
